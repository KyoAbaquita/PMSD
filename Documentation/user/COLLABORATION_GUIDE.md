# Real-Time Features

## Activity Feed

- Live updates when:
  - Tasks change status
  - New comments are posted
  - Files are uploaded
  - **Risk log entries** _(Sprint 5)_
- **Report generation** _(Sprint 5)_

## Notifications

- Email alerts for:
  - Overdue tasks
  - High-severity risks _(New in Sprint 5)_
